# LEARNING_PATH.md

## Propósito

Este archivo orienta el aprendizaje técnico necesario para construir APP DEPORTE con el stack oficial del proyecto.

## Ruta Recomendada

### 1. Producto y Dominio

- Entender el flujo operativo de un propietario de cancha.
- Mapear estados de reserva y pago.
- Identificar reglas de aislamiento multi-tenant.

### 2. Frontend

- Next.js App Router.
- React con TypeScript.
- TailwindCSS.
- Shadcn UI.
- Formularios y validación de entrada.

### 3. Backend

- Node.js con TypeScript.
- Fastify.
- Clean Architecture.
- Repository Pattern.
- Validación de requests.

### 4. Supabase

- Supabase Auth.
- Supabase PostgreSQL.
- Row Level Security.
- Supabase Storage.
- Supabase Realtime.

### 5. Calidad

- TypeScript estricto.
- Pruebas unitarias.
- Pruebas de integración para casos críticos.
- Separación de responsabilidades.
